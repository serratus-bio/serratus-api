# serratus.io

Serratus website built using Vue.js.

## Quickstart

Clone the repository, build and preview the site locally:

```
git clone https://github.com/serratus-bio/serratus.io.git
cd serratus.io
npm install
npm run serve
```

Go to http://localhost:8080/
